# 快速开始指南

## 🎯 立即运行

### 方法1：Android Studio
1. 打开 Android Studio
2. 选择 "Open" 打开项目目录
3. 等待 Gradle 同步完成
4. 点击 Run 按钮（绿色三角形）

### 方法2：命令行
```bash
cd /storage/emulated/0/AndroidIDEProjects/xbjsb
./gradlew installDebug
adb shell am start -n com.example.xbjsb/.MainActivity
```

## 🎨 体验要点

### 首次启动
1. 看到空状态提示
2. 点击"写第一篇日记"按钮
3. 填写标题和内容
4. 选择心情
5. 添加标签（可选）
6. 点击右上角的 ✓ 保存

### 浏览日记
- 在列表页滑动查看所有日记
- 点击卡片查看详情
- 点击 ❤️ 收藏日记
- 向下滚动时 FAB 自动隐藏

### 搜索和筛选
1. 点击顶部 🔍 图标展开搜索栏
2. 输入关键词搜索
3. 点击筛选图标选择心情
4. 点击 ❤️ 图标只看收藏
5. 点击"清除"重置所有筛选

### 编辑和删除
- 在详情页点击 ✏️ 编辑
- 修改后点击 ✓ 保存
- 点击 🗑️ 删除（会有确认对话框）

## 🐛 常见问题

### 应用崩溃？
- 检查 Gradle 同步是否完成
- 清理项目：Build → Clean Project
- 重新构建：Build → Rebuild Project

### 看不到动态配色？
- 需要 Android 12+ (API 31+)
- 在设置中设置壁纸主题色

### 列表为空？
- 这是正常的，首次启动没有数据
- 点击浮动按钮创建第一篇日记

## 📝 使用技巧

### 高效写日记
1. 使用有意义的标题
2. 选择准确的心情
3. 添加关键标签方便检索
4. 重要的日记记得收藏

### 快速查找
- 搜索支持标题、内容、标签
- 使用心情筛选回顾特定时期
- 收藏筛选快速找到重点内容

### 数据安全
- 所有数据存储在本地
- Room 数据库自动管理
- 删除操作有二次确认
